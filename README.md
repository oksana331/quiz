# OpenTest Quiz

Проект для открытого урока без базы данных.

## Что есть
- несколько вопросов
- в каждом вопросе один правильный ответ
- страница ученика
- отдельная панель учителя
- отдельный график по каждому вопросу
- работа без MySQL

## Запуск
1. Открой проект в IntelliJ IDEA
2. Дождись загрузки Gradle
3. Запусти `OpenTestApplication`
4. Открой:
   - http://localhost:8080/test
   - http://localhost:8080/teacher
